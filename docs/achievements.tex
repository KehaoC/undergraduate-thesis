  \item Wentao Wan, Zhuojie Yang, Yongcan Chen, Chenglin Luo, Ruilin Wang, \textbf{Kehao Cai}, Nan Kang, Liang Lin, Keze Wang.
        \textit{SR-FoT: A Syllogistic-Reasoning Framework of Thought for Large Language Models Tackling Knowledge-based Reasoning Tasks}.
        In \textit{Proceedings of the Thirty-Ninth AAAI Conference on Artificial Intelligence (AAAI 2025)}.
        arXiv:2501.11599.

        \smallskip
        \noindent\textbf{内容简述：}本文提出多阶段三段论推理框架（SR-FoT），模拟人类演绎推理范式，引导大语言模型依次构建大前提、小前提并推导最终结论，在 ScienceQA、StrategyQA、BoolQ 等知识推理基准上显著优于标准思维链方法。该工作与本论文在核心认知上高度呼应：SR-FoT 揭示了结构化推理路径对 LLM 不确定性收敛的关键作用——恰如本论文所主张的，Skill 的本质价值正是为 Agent 的下一词元预测提供高确定性的结构化先验，将 LLM 的无限不确定性坍缩为领域内的确定性行动路径。
\end{theachievements}
