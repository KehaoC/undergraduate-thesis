# 中山大学本科毕业论文项目

## 项目概述
- 基于 sysu-thesis LaTeX 模板的本科毕业论文
- 使用 XeLaTeX 编译，`make main` 生成 PDF
- 代码和论文共存于同一仓库

## 重要标记
- `template-original` tag：保存了模板原始内容（含教程和示例），查看方式：`git show template-original:<文件路径>`

## 项目结构
- `main.tex` — 主入口
- `sysusetup.tex` — 个人信息配置
- `docs/` — 各章节内容
- `reference.bib` — 参考文献
- `image/` — 图片
- `code/` — 代码

当出现各种编写论文或者 latex 格式相关的问题时，请参考 sysuthesis-guide.tex 文件。
